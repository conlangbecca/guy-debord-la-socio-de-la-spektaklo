# guy-debord-la-socio-de-la-spektaklo
Esperanta traduko de "La société du spectacle" de Guy Debord

La inkluzivita franca originalo estas publika havaĵo laŭ volo de la aŭtoro.
La inkluzivita angla traduko estas tiu de Ken Knabb, kiu estas publika havaĵo laŭ volo de la tradukisto.
Ankaŭ ĉi tiu Esperanta traduko estas publika havaĵo laŭ mia propra volo.
